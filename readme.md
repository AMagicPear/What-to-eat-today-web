# 今天吃什么
这是一个我专门为我妹妹写的用来帮助决定今天吃什么的Web。
现在包括的功能如下：
- [x]  可保存多个食物，并分别设置其早上、中午、晚上时间段的权重，点击“今天吃什么”按钮，系统将根据当前时间段各个食物的权重随机选择一项。
- [ ] 每种食物在被连续选择两次后的下一次不会再次被选择。
- [ ] 会优先选择近30次选择内从未被选中的食物。
